#2. MSBBS란?
  APACHE,PHP,MySQL 기반에 만들어진 웹보드로서, 기본기능(글쓰기,글보기)외에
 편리성을 위주로 하는 기능을 더 첨부한 보드입니다.


#3. 라이센스

 1. 이 소스를 사용하는 것은 이 라이센스에 동의하는 것으로 간주합니다.
 2. 이 소스는 배포, 사용에 아무런 제한이 없는 오픈소스입니다.
 3. 이 소스를 사용함에 따라 발생하는 피해에 대해서 원제작자는 책임이 없습니다.
 4. 스킨을 제작할 경우 그 제작자는 그 스킨에 대해 권한을 가집니다.
 5. 제작자는 유지, 보수, 개발의 의무가 없습니다.


#4. 주의하실 점

 1. 반드시 공식배포처에서 반드시 최신버전을 확인하시어 설치하십시오.
 2. 설치하신 후에도 각종 패치나 업그레이드에 신경써주십시오.
 3. 관리자 비밀번호 관리를 소흘히하지 마십시오.


#5. 간략한 설치방법

 1. lib.php 파일을 열어 최상단에 수정할 부분을 수정한다.
 2. msbbs 파일들을 ftp 로 자신의 계정에 전송한다.
 3. 자료실 기능을 사용할 것이라면 data 폴더의 퍼미션을 707 로 조정한다.
 4. 웹 상에서 install.php 파일을 불러온다.
 5. 설치를 한다.
